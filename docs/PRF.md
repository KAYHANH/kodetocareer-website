# Product Requirements File (PRF)

# Project Name
KodeToCareer 2.0

## Vision

KodeToCareer is a premium AI-powered EdTech platform that helps students learn in-demand skills, build real-world projects, secure internships, obtain placements, and explore study-abroad opportunities.

The platform should feel closer to Linear, Stripe, Coursera, and Vercel than a traditional coaching institute.

---

# Target Users

- College Students
- Fresh Graduates
- Working Professionals
- Study Abroad Aspirants
- Recruiters
- Corporate Partners

---

# Goals

- Increase student enrollments
- Generate qualified leads
- Build trust through placements and success stories
- Offer AI-powered career guidance
- Provide a seamless learning experience
- Create a scalable education platform
